# 🎓 Benny's Learning Portfolio (學習型專案)

> **這不僅僅是一個作品集，這是一個「教你如何寫出作品集」的互動式教材。**

這個專案經過重構，旨在教導初學者如何從零打造一個現代化、具備玻璃擬態 (Glassmorphism) 風格的網頁。

## 📚 如何開始學習？

1.  **閱讀 [TUTORIAL.md](./TUTORIAL.md)**：這是本專案的教科書，詳細解說了學習路徑與挑戰。
2.  **查看原始碼**：
    *   `index.html`: 包含詳細的 HTML 語意化標籤教學。
    *   `css/style.css`: 包含詳細的 CSS 變數、Flexbox、Grid 與玻璃擬態教學。
    *   `js/script.js`: 包含詳細的 DOM 操作與事件監聽教學。

## 🌟 關於這個專案

作為一名程式開發初學者，我希望透過這個網頁來記錄我的成長軌跡。本網頁採用了現代化的 **Glassmorphism (玻璃擬態)** 風格設計，旨在打造一個視覺效果出眾且互動流暢的展示空間。

### 🛠 使用技術
- **前端架構**: HTML5, CSS3 (Bootstrap 5)
- **互動邏輯**: JavaScript (jQuery)
- **視覺風格**: Glassmorphism UI, Animate.css
- **字體設計**: Google Fonts (Outfit)

## 💡 開發筆記與 AI 的角色

在建構這個作品集網頁的過程中，我積極嘗試了 **AI 協作開發**。
- **UI/UX 設計**: 我利用 AI 工具輔助構思玻璃擬態的配色與排版，大幅提升了開發效率。
- **學習過程**: 對於 AI 生成的程式碼（如 CSS 漸層動畫與 JS 事件監聽），我皆經過深度閱讀與理解，確保自己能完全掌握其運作原理與後續維護。

我認為**「善用現代工具解決問題」**也是工程師的重要技能之一。我熱衷於在實戰中學習，並享受不斷優化程式碼的過程。
