A modern, high-performance portfolio website built with **Vanilla HTML, CSS, and JavaScript**.

### Key Features
- **Zero Dependencies**: No Bootstrap, No jQuery, No Frameworks. Just pure, fast web technologies.
- **Glassmorphism**: Premium frosted glass effects using `backdrop-filter` and multilayered shadows.
- **Fluid Typography**: Text scales smoothly across all devices using CSS `clamp()`.
- **Micro-interactions**: Hover lifts, button shine effects, and smooth scroll animations.
- **Dynamic Backgrounds**: Animated aurora blobs that move gently behind the content.
