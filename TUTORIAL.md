# 🎓 專案學習指南：打造個人的現代化作品集

歡迎來到這個學習專案！這個專案不僅僅是一個網頁，更是一本「活生生」的教科書。
我們已經將原始程式碼轉化為教學素材，帶你一步步理解現代網頁開發的核心概念。

## 📚 你將學到什麼？

1.  **HTML5 語意化結構**：如何寫出搜尋引擎 (SEO) 和輔助科技都能讀懂的網頁。
2.  **CSS3 現代排版**：
    *   **CSS Variables (變數)**：如何一鍵切換整站色調。
    *   **Flexbox & Grid**：如何打造響應式佈局 (Responsive Layout) 和 Bento Grid（便當盒式佈局）。
    *   **Glassmorphism (玻璃擬態)**：如何製作半透明、模糊背景的現代感卡片。
3.  **JavaScript 互動邏輯**：
    *   **DOM 操作**：如何讓網頁元素動起來。
    *   **Event Listeners**：如何監聽滑鼠移動、點擊和滾動事件。
    *   **非同步模擬**：如何用 `setTimeout` 模擬從伺服器抓取資料的感覺。

## 🚀 學習路徑

請依照以下順序閱讀程式碼檔案，我已經在程式碼中加入了詳細的中文註解（就像你的私人導師一樣）：

### 第一站：`index.html` (網頁的骨架)
打開 `index.html`，你將看到：
*   `<meta>` 標籤如何幫助你的網站被 Google 搜尋到。
*   `<nav>`, `<header>`, `<main>`, `<footer>` 這些標籤的意義。
*   如何引入外部資源 (字體、圖示庫)。

### 第二站：`css/style.css` (網頁的皮膚)
打開 `css/style.css`，重點關注：
*   `:root`：定義全域變數的地方（試著改改看顏色！）。
*   `.glass-card`：玻璃擬態的核心魔法 (`backdrop-filter`)。
*   `.bento-grid`：如何用 Grid 系統切出漂亮的網格。

### 第三站：`js/script.js` (網頁的肌肉)
打開 `js/script.js`，尋找這些邏輯：
*   **滑鼠跟隨效果**：計算滑鼠位置並更新 CSS 變數，讓卡片有光暈效果。
*   **數字捲動動畫**：如何讓數字從 0 跑到 500。
*   **API 模擬**：按下 "Sync" 按鈕時發生了什麼事？

## 🛠️ 實戰挑戰 (動手做做看)

光看是不會學會的，試著完成以下挑戰：

1.  **初級挑戰**：
    *   打開 `css/style.css`，找到 `:root` 區域，將 `--accent-orange` 的顏色代碼改成你喜歡的顏色（例如紫色 `#a855f7`）。重新整理網頁，看看發生了什麼？

2.  **中級挑戰**：
    *   在 `index.html` 的 `bento-grid` 區域中，複製一個 `.bento-item`，試著加入第四個作品卡片。

3.  **高級挑戰**：
    *   修改 `js/script.js` 中的 `mockData` 陣列，把 "Repo" 的名字改成你自己的專案名稱。

## 💡 常見問題

**Q: 為什麼這網頁看起來這麼「現代」？**
A: 因為它使用了「玻璃擬態 (Glassmorphism)」和「深色模式 (Dark Mode)」的設計語言，加上細微的動畫互動（如滑鼠移動時的光暈）。

**Q: 我需要安裝什麼軟體嗎？**
A: 不需要！只要有 VS Code 和瀏覽器（Chrome/Edge）即可。建議安裝 "Live Server" 擴充套件，可以即時看到修改結果。
