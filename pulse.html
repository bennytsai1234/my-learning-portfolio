Pulse

Modern Android Music Player

Kotlin Jetpack Compose Material 3 Room

Sonic Perfection

Pulse is not just an MP3 player; it's a visual and auditory experience tailored for modern Android devices. Built completely with Jetpack Compose, it features a fluid, gesture-driven interface that adapts to your music's album art in real-time.

Key Features

Dynamic Theme

UI colors automatically extract and adapt from the current song's album artwork using Palette API.

Waveform Viz

Real-time audio visualization rendering directly on the canvas for a responsive, immersive feel.

Synced Lyrics

Support for .lrc files with smooth scrolling and time-synced highlighting.

5-Band EQ

Built-in equalizer with bass boost and virtualizer to tune the sound exactly how you like it.

@HiltViewModel class MainViewModel @Inject constructor( private val musicRepository: MusicRepository, private val settingsRepository: SettingsRepository ) : ViewModel() { val uiState = musicRepository.songs .map { UiState.Success(it) } .stateIn(viewModelScope, SharingStarted.Eagerly, UiState.Loading) }

Under the Hood

  • Clean Architecture
    Strict separation of concerns into Data, Domain, and Presentation layers.
  • MVVM Pattern
    Unidirectional Data Flow (UDF) with StateFlow and ViewModels.
  • Media3 / ExoPlayer
    Robust media playback handling including foreground services and notification controls.
  • Dependency Injection
    Powered by Hilt for testability and modularity.

Engineered for Performance

Pulse is built with scalability in mind. By leveraging the latest Android development standards, the codebase remains maintainable while delivering 60fps animations even on mid-range devices.

Offline-first capability is ensured by Room Database, caching metadata and album art for instant loading times.
